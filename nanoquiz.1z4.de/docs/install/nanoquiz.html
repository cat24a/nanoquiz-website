Program można pobrać z platformy GitHub lub za pomocą przycisku poniżej.

Instalacja

  1. Utwórz nowy, pusty folder. Wybierz dla niego nazwę, po której łatwo go rozpoznasz, np. „nanoquiz”. Ten folder będzie później nazywany „folderem programu”.
  2. Umieść pobrany plik w folderze programu.
  3. Uruchom pobrany plik.

Na ekranie powinno pokazać się okno programu z napisem BRAK QUIZÓW. Jeśli tak jest, instalacja programu udała się. Jeśli nie, oznacza to, że zrobiłeś/zrobiłaś coś nie tak. (np. nie zainstalowałeś/zainstalowałaś javy zgodnie z poradnikiem) Jeśli wciąż masz problem, możesz wysłać do mnie e–mail.

Możesz zauważyć, że w folderze programu został utworzony folder o nazwie „quiz”. W nim należy umieścić pliki quizów.
